# 🏢 건물 부수기 게임 🗡️

위에서 떨어지는 비트코인 건물을 부수는 액션 게임입니다!

## 🎮 게임 방법

### 조작법
- **점프**: 키보드 방향키 위(↑) 또는 화면 왼쪽의 점프 버튼
- **베기**: 스페이스바 또는 화면 오른쪽의 베기 버튼

### 게임 규칙
- 위에서 비트코인 건물이 떨어집니다
- 점프하면 건물을 1층 정도 위로 밀 수 있습니다
- 베기로 건물에 데미지를 줍니다 (베기당 데미지 1)
- 베기는 왼쪽, 오른쪽 순서로 번갈아가며 실행됩니다
- 건물은 10층으로 구성되어 있으며, 각 층마다 체력이 3입니다
- 건물에 깔리면 게임 오버입니다!

### 난이도
- 10층짜리 건물을 완전히 부수면 다음 건물이 떨어집니다
- 건물을 부술 때마다 다음 건물의 1층당 체력이 +2씩 증가합니다
- 점점 더 강해지는 건물을 계속 부숴보세요!

### 점수
- 건물에 데미지를 줄 때마다 점수가 올라갑니다
- 게임 종료 시 최고 점수 순위(TOP 5)가 표시됩니다
- 점수는 브라우저의 로컬 스토리지에 저장됩니다

## 🖼️ 캐릭터 이미지 설정

게임을 제대로 즐기려면 캐릭터 이미지를 추가해야 합니다:

1. `images` 폴더에 다음 이미지 파일을 추가하세요:
   - `sol.png` - 기본 캐릭터 이미지
   - `sol_left.png` - 왼쪽으로 베기 동작 이미지
   - `sol_right.png` - 오른쪽으로 베기 동작 이미지

2. 이미지가 없어도 게임은 플레이 가능하지만, 플레이스홀더 그래픽이 표시됩니다.

## 🚀 실행 방법

### 로컬에서 실행
1. 이 저장소를 클론하거나 다운로드합니다
2. `index.html` 파일을 브라우저로 엽니다
3. 게임을 즐기세요!

### GitHub Pages로 배포
1. GitHub 저장소에 푸시합니다
2. Settings > Pages로 이동합니다
3. Source를 "main" 브랜치로 설정합니다
4. 몇 분 후 제공되는 URL로 접속하여 게임을 플레이할 수 있습니다
