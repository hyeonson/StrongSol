# 📚 업무 부시기 게임 (StrongSol) - 프로젝트 문서

---

## 1. 프로젝트 개요

### 프로젝트 이름
**업무 부시기 게임 (Strong Sol)**

### 결과물 형태
🎮 **웹 게임**

### 한 줄 요약
**하늘에서 떨어지는 업무(스트레스, 민원, 숙취 등)를 점프와 베기로 파괴하며 살아남는 2D 액션 게임**

---

## 2. 프로젝트 목적

### 재미·흥미 제공
- 🎯 **스트레스 해소**: 업무, 스트레스, 민원 등을 직접 부수며 카타르시스 제공
- 🎮 **엔터테인먼트**: 간단하지만 중독성 있는 게임플레이
- 🏆 **도전 과제**: 난이도별 최고 점수 경쟁
- 💪 **성취감**: 강력한 검 구매 및 필살기 시스템

## 3. 주요 기능

### 3.1 핵심 전투 시스템 ⚔️
- **점프** (↑ / 점프 버튼): 건물을 2층 높이 위로 밀기
- **베기** (Q / 베기 버튼): 3층 높이 범위 내 공격
- **일격 필살기** (R / 일격 버튼): 화면 위까지 날아가 모든 업무 파괴
  - 베기 성공 시 게이지 충전 (난이도별 20/30/40회)

### 3.2 상점 시스템 🛒
- **짱센검** (200점): 공격력 10배 증가
- **전설의 검** (2000점): 공격력 100배 증가
- 게임 중 B 키로 상점 접근 가능
- 일시정지 기능 포함

### 3.3 난이도 선택 🎯
- **Easy**: 체력 증가 +2, 일격 게이지 20회, 낙하 속도 2.5
- **Normal**: 체력 증가 +4, 일격 게이지 30회, 낙하 속도 3.5
- **Hard**: 체력 증가 +6, 일격 게이지 40회, 낙하 속도 5.0
- 난이도별 독립적인 점수 기록
- 난이도가 높을수록 건물이 빠르게 떨어져 긴박감 증가

### 3.4 시각/청각 효과 🎨🔊
- **6가지 랜덤 색상 테마**: 비트코인, 에메랄드, 사파이어, 루비, 자수정, 토파즈
- **파티클 시스템**: 
  - 데미지 시 작은 파티클 (8개)
  - 층 파괴 시 대형 파티클 (67개)
  - 회전, 반짝임, 페이드 아웃 효과
- **효과음**:
  - 베기 소리 (칼 휘두르는 소리)
  - 폭발 소리 (층 파괴 시)

### 3.5 다양한 업무 타입 📋
- 랜덤 업무: 업무, 스트레스, 민원, 숙취
- 책/서류 스타일 디자인
- 페이지 라인, 금색 모서리 장식
- 난이도 증가 시스템

### 3.6 카메라 시스템 📹
- 플레이어 추적 카메라
- 부드러운 카메라 이동
- 패럴랙스 스크롤링 (구름)

- **그라데이션** - 하늘, 건물 그림자

- **터치 이벤트** - 모바일 컨트롤
- **반응형 디자인** - 다양한 화면 크기 지원
- **제스처 방지** - 줌, 스크롤 차단

## 5. 사용자 흐름 (User Flow)

### 5.1 게임 시작
```
1. 게임 로드
   ↓
2. 시작 화면
   - 게임 설명 확인
   - 조작법 학습
   ↓
3. 난이도 선택
   - Easy / Normal / Hard 중 선택
   ↓
4. "게임 시작" 버튼 클릭
```

### 5.2 게임 플레이
```
5. 게임 화면 진입
   - 캐릭터: 화면 중앙 하단
   - 업무: 화면 위에서 떨어짐
   ↓
6. 업무 파괴
   - 점프(↑): 업무 밀기
   - 베기(Q): 업무 공격
   - 일격(R): 필살기 (게이지 가득 찰 때)
   ↓
7. 점수 획득
   - 베기 성공: 공격력만큼 점수
   - 일격 성공: 파괴된 총 HP만큼 점수
```

### 5.3 상점 이용
```
8. 상점 접근 (B 키 또는 상점 버튼)
   ↓
9. 아이템 구매
   - 짱센검 (200점): 공격력 10배
   - 전설의 검 (2000점): 공격력 100배
   ↓
10. 상점 나가기
    - 게임 재개
```

### 5.4 게임 오버 & 재시작
```
11. 게임 오버 조건
    - 지면에 있을 때 업무가 머리 위 24픽셀 이내 접근
    ↓
12. 게임 오버 화면
    - 최종 점수 확인
    - 해당 난이도 최고 점수 순위 확인
    ↓
13. 재시작
    - "다시 하기" 버튼
    - 시작 화면으로 복귀
```

## 💬 전체 프롬프트 히스토리

### Phase 1: 기본 게임 메커니즘 구축

**1. 충돌 시스템 구현**
```
건물이랑 캐릭터랑 겹칠 수 없어. 
그리고 베는 동작은 항상 할 수 있어. 
단지 건물과 닿아야 데미지를 줄 수 있는거야. 
그리고 화면 왼쪽에 점프 버튼, 오른쪽에 베기 버튼이 없어.
```

**2. 캐릭터 스펙 조정**
```
캐릭터 크기를 더 키워줘. 
그리고 점프 거리도 2배로 키워줘.
```

**3. 게임 오버 조건 설정**
```
게임 종료되는 시점은 캐릭터가 지면에 닿았을 때, 
머리가 건물에 닿으면 종료야.
```

### Phase 2: 게임플레이 개선

**4. 점프 시스템**
```
점프를 한번하면 한번밖에 뛰지 못해. 
그리고 건물 부시면, 5층 사이 간격으로 새로운 건물이 또 나와. 
그리고 건물 떨어지는 속도도 올려줘.
```

**5. 베기 쿨다운 제거**
```
왜 베기가 연속으로 안되지??
```

**6. 게임 오버 조건 완화**
```
게임 종료되는 조건을 플레이어 머리 위 1.1배 높이 범위 내일 때로 수정해줘
→ 0.5로 바꿔줘
→ 0.2로 바꿔줘
```

**7. 베기 범위 조정**
```
베기가 성공하려면 현재 캐릭터의 위치의 가장 위 y 보다 
+ 2층만큼 건물이 있으면 부실 수 있어. x축은 상관없어.
→ 3층 높이 범위 내로 변경
```

**8. 건물 크기 조정**
```
건물을 좌우로 넓게 그려줘 화면 가로길의 1/3 정도로
→ 2/3로 변경
```

### Phase 3: 상점 및 업그레이드 시스템

**9. 상점 시스템 구현**
```
베기 버튼 바로 위에 상점 버튼도 추가해줘. 
상점 버튼을 누르면 게임이 일시정지되고, 
'짱센검'을 현재 얻은 점수의 100점을 차감해서 구매할 수 있어.
구매하면 캐릭터 이미지가 sol_1.png로 바뀌고, 검의 공격력은 10으로 바뀌어.
```

**10. 짱센검 가격 조정**
```
짱센검 가격은 100이 아니라 200으로 바꿔줘.
```

**11. 전설의 검 추가**
```
상점에 짱센검 아래에 전설의 검을 추가해줘. 
전설의 검은 2000점으로 구매할 수 있어. 
검의 공격력은 100이야.
```

### Phase 4: 필살기 시스템

**12. 일격 필살기 구현**
```
필살기 기능을 추가할꺼야. 
일격 버튼은 키보드 R 버튼을 눌러도 동작할 수 있어. 
베기가 20번 성공하면 버튼이 활성화 되고, 
일격버튼을 누르면 캐릭터가 화면 가장 위까지 점프하고, 
캐릭터에 닿는 모든 건물의 층을 한번에 부셔.
```

**13. 필살기 개선**
```
일격을 사용하면 화면 가장 위까지만 올라가고, 
화면에 보이는 건물만 파괴할 수 있어.
```

**14. 일격 파괴 로직 변경**
```
일격을 이용하면 건물을 통째로 부시는게 아니라 
화면에 보이는 건물의 일부만 부실 수 있어.
```

### Phase 5: UI/UX 개선

**15. 키보드 단축키 변경**
```
베기 버튼은 스페이스바가 아니라 키보드 Q 누르면 벨 수 있어. 
그리고 상점버튼 키보드 B 버튼을 누르면 동작해.
```

**16. 모바일 최적화**
```
모바일에서도 게임할 수 있게 변경해줘
```

**17. 버튼 크기 조정**
```
점프버튼 왼쪽 최하단에 위치 시켜주고, 
점프버튼과 베기버튼 크기 2배로 바꿔줘
```

**18. 상점 UI 개선**
```
상점버튼 누를 때, 점수, 현재 건물도 점프, 베기 버튼처럼 같이 사라지고, 
다시 게임 화면으로 돌아가면 보여줘
```

### Phase 6: 시각/청각 효과

**19. 효과음 추가**
```
베기 버튼 누를 때, 효과음을 넣고 싶은데, 칼 부딪히는 소리 넣어줘
```

**20. 건물 디자인**

```
건물을 책이나, 서류같이 꾸며줘
```

```
새로운 건물이 떨어질 때 마다, 색상도 랜덤하게 바꿔줘.
```

```
건물 부실 때 파티클 효과 추가해줘.
```

```
건물이 한층씩 부셔질 때마다 폭탄 터지는 소리 넣어줘.

```


**28. 화면 제한**
```
점프했는데 화면에 건물이 없으면, 캐릭터가 화면 밖으로 나갈 수 없어.

```

**29. 충돌 처리 개선**
```
점프할때 캐릭터가 건물 위로 올라가버려. 
건물 하단에 막혀서 올라가면 안돼.
```

### Phase 8: 난이도 및 점수 시스템

**30. 난이도 시스템**
```
현재 난이도는 easy, normal, hard 모드 중에 easy 모드야.

난이도가 높아질 수록, 다음에 생성되는 건물의 체력이 증가폭이 더 커져. 
그리고 일격 게이지도 더 많이 필요해.
→ 난이도가 어려워질수록 건물 떨어지는 속도도 빠르게 해줘
```
